# Production Checklist

## Code Quality
- [ ] All tests pass (`npm test`).
- [ ] No console warnings in tests.
- [ ] `BUGS.md` updated or cleared.

## Documentation
- [ ] `README.md` is up to date.
- [ ] `docs/` folder is populated (00-11).
- [ ] API changes documented in `08_API_AND_INTEGRATION.md`.

## Release
- [ ] Version bumped in `package.json`.
- [ ] Changelog updated.
- [ ] Git tag created.
